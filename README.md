# 🚛 Sistema de Control de Llantas

## Descripción

Sistema web desarrollado para la gestión y control de llantas de una flota de transporte.

Permite registrar calibraciones, cambios de llantas, inventario, historial por unidad y generación de reportes operativos.

---

## Problema que resuelve

Antes del sistema, el control de llantas se realizaba manualmente mediante registros físicos y hojas de cálculo dispersas.

Esto dificultaba:

* Seguimiento del historial de llantas
* Control de inventario
* Registro de calibraciones
* Generación de reportes
* Trazabilidad de movimientos

---

## Funcionalidades

### Gestión de inventario

* Registro de llantas
* Control de existencias
* Identificación por código TM

### Calibraciones

* Registro de PSI
* Registro de profundidad
* Historial por unidad

### Cambios de llantas

* Llanta desmontada
* Llanta montada
* Motivo del cambio
* Ubicación posterior

### Reportes

* Reportes automáticos
* Historial por unidad
* Control operativo
